A small dev-tools shop. Releases ship Tuesday mornings, twice a month.

v0.7.4

2026-04-29
  • Add --watch flag to codeline build for incremental rebuilds (closes #412).
  • Add JSON output mode to codeline lint; pipe-friendly for CI.
  • Fix tsconfig.json resolution failing when extends points outside the workspace root.
  • Fix incorrect exit code (0 instead of 1) when codeline lint finds errors but no warnings.
  • Deprecate --legacy-loader; will be removed in v0.9.

v0.7.3

2026-04-15
  • Fix race condition in the file-watcher when many small files change simultaneously (e.g., during a git checkout).
  • Fix the loader silently swallowing parse errors in postcss plugin chains.
  • Add codeline doctor — a diagnostic command that checks node version, lockfile parity, and config validity.
